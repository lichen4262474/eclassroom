# eclassroom
  E-classroom is a digital learning platform  that allows teachers to share learning resources, grade assignments, communicate with students and parents, and perform grade analysis. 
  It allows the students to check the shared resources, submit assignments, communicate with their teachers and perform grade analysis. 
#Teachers can…
  Create and modify personal profile;
  Create,modify,delete a class;
  Share,modify,delete an announcement;
  Share,modify,delete lessons;
  Share,modify,delete and grade assignments;
  Unenroll students;
  Contact students and parents;
  Perform grade analysis with dynamic graph about students’ grades and assignments status.
#Students can…
  Create and modify personal profile;
  Join and drop a class;
  Check shared learning resources;
  Check and submit assignments;
  Contact teachers;
  Perform grade analysis with dynamic graph about student’s grades and assignments status.
#Technology Stack for this application:
  Front end: html5, css3, javascript, bootstrap5;
  Database: MySQL;
  BackEnd: java, Spring boot, Spring JPA, Spring Security;
  Template Engine: Thymeleaf;
